Important FAQ

What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each node corresponds to a part of the document, such as elements, attributes, and text.

How do you select an element from hte DOM

Use document.querySelector() or document.getElementById(). ETC

What is event delegation in the context of the DOM , and why is it useful?

Event delegation is handling events on parent elements, improving performance and simplifying code by reducing the number of event listeners

How do you manipulate an element's attributes and styles using hte DOM?

Use setAttribute(), getAttribute(), and style.property to manipulate attributes and styles in the DOM.
